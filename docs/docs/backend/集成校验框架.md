# 集成校验框架

- 添加依赖

```java
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-validation</artifactId>
        </dependency>
```

- 添加校验

```java
@Data
public class MemberReq {
    @NotBlank(message = "手机号不能为空")
    String phone;

    String name;

}
```

```java
    @PostMapping("/register")
    public ApiResponse<Integer> register(@RequestBody @Valid MemberReq memberReq){
        return ApiResponse.success(memberService.register(memberReq));
    }
```
